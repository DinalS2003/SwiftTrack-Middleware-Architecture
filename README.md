# SwiftTrack Middleware Architecture

Enterprise middleware logistics platform developed using microservices architecture.

## Features
- API Gateway integration
- REST API communication
- RabbitMQ asynchronous messaging
- PostgreSQL database integration
- Dockerized services

## Technologies
- Flask
- FastAPI
- RabbitMQ
- PostgreSQL
- Docker
- HTML/CSS/JavaScript

## Components
- API Gateway
- CMS Adapter
- ROS Adapter
- WMS Adapter
- Order Service
- Client Portal
- Driver App
